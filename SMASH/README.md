# SMASH.

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# fix everything
$ npm run lint:fix

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
